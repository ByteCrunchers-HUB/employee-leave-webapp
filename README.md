# Employee Leave Webapp

Node + MongoDB backend with HTML/CSS/JS + Tailwind frontend (no React).

## Setup
1. Add your MongoDB Atlas URI in `.env` (`MONGODB_URI`).
2. Keep `MONGODB_DB=EMPleave` (or set your own DB name).
3. Install: `npm install`
4. Start: `npm start`
5. Open `http://localhost:5000`

Default admin is auto-created on first run:
- admin@company.com / admin123
